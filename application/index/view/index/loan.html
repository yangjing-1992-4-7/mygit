第二部分：本协议相关具体条款
第一条 释义

在本合同中，除非上下文另有解释，下列词语具有以下含义：

1.1
本协议：指本《借款服务协议》及所有附件中的任何条款、明细和信息。
1.2
借款人：本协议甲方，具有完全民事权利能力和民事行为能力，能独立承担本协议项下权利义务的自然人或其他法律实体。
1.3
出借人：本协议乙方，具有完全民事权利能力和民事行为能力，能独立行使和承担本协议项下权利义务的自然人或其他法律实体。
1.4
债权：指本协议中出借人拥有的全部权益（包括本金、利息、罚息和违约金），债权以人民币计价。
1.5
支付机构：指在本协议甲乙双方之间作为中介机构提供资金冻结、转移服务的第三方支付机构或银行。
1.6
资金冻结：乙方投标后，将出借资金汇入在监管账户项下的乙方第三方支付机构账户后，资金将被冻结；甲方还款时，将还款资金汇入在监管账户项下的甲方第三方支付机构账户后，资金将被冻结。
1.7
工作日：指中华人民共和国法律规定的工作日（法定节假日除外）。
1.8
提前还款：本协议附件中约定了借款人的本息偿还日期和具体金额等相关还款计划，借款人可能在协议规定的偿还日期结束前，在某一期将剩余本息提前偿还给乙方，从而使乙方的资金比约定的计划提前收回。
第二条 借款流程
2.1
发布借款需求：由甲方向丙方提出申请，经丙方形式审核后在平台官网上发布借款需求，该借款需求，构成借款合同内容的一部分。
2.2
出借资金冻结：乙方点击“立即出借”按钮即视为其已经向丙方发出不可撤销的授权指令，授权丙方委托第三方支付机构及其设立的以丙方名义开立、资金独立于丙方自有资金的监管账户（“监管账户”）的监管银行等合作机构（“存管人”），在监管账户中乙方关联平台用户名下的账户（“乙方第三方账户”），冻结金额等同于本协议第一部分“借款本金”的资金。
2.3
本协议生效：本协议成立的同时，甲方即不可撤销的授权丙方委托存管人，将金额等同于本协议第一部分“借款本金”的资金，由监管账户项下乙方第三方账户中划转至监管账户项下甲方第三方账户下，划转完毕即视为本协议生效。
第三条 本息偿还方式
3.1
甲方到期一次性还本还息的方式向乙方还款。甲方到期还款总金额见本协议附件还款计划所示。
3.2
甲方授权丙方委托相应的第三方支付机构及监管银行等合作机构，按本协议附件一所述还款计划将金额等同于乙方当期应收金额的资金，由监管账户项下甲方第三方账户划转至监管账户项下乙方第三方账户，划转完毕即视为本期还款发放成功。
3.3
甲方应保证在每期还款当日或之前的甲方第三方账户中资金余额足够支付当期应还利息、应还本金及账户管理费。
3.4
如果还款日遇到法定假日或公休日，还款日期不顺延。
3.5
甲方如提前还款，将按照实际使用的天数支付相关利息。
第四条 逾期罚息

若甲方晚于本协议附件一所述还款计划规定的还款日还款，应向乙方支付逾期罚息。罚息计算方法如下：

4.1
逾期罚息：自逾期之日始，每日按尚未清偿的全部剩余本金和利息的万分之二的费率按日计收罚息，直至清偿完毕之日；因逾期导致的账户管理费另计。
4.2
若甲方偿还金额不足，偿还顺序按照先后顺序为账户管理费、逾期罚息、应还利息、应还本金。
4.3
乙方保留将甲方违约失信的相关信息在媒体等相关机构披露的权利。因甲方未还款而带来的尽职调查费、诉讼费用、保全费、律师代理费等所有实现债权的费用将由甲方承担。

（8）甲、乙双方在丙方平台交易成功的，丙方有权收取平台中介费:

（a）以30日为一个借款期，在一个借款期内，平台收取的中介费为：基础中介费及实际使用借款中介费的总和；

  2）实际使用借款中介费的收费标准为：甲、乙交易金额的十万分之八乘以实际使用天数。

（b）如借期为一个借款期的倍数，平台收取的中介费为：基础中介费及实际使用借款中介费的总和

  1）基础中介费的收费标准为：甲、乙交易金额的万分之五乘以借款期倍数；

  2）实际使用借款中介费的收费标准为：甲、乙交易金额的十万分之八乘以实际使用天数。
